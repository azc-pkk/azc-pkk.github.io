普普通通的开发日志。

前言

本来想写一篇用 搭建个人博客的教程的，但是看了很多大佬的博客后感觉自己不配，所以改成了网站建设记录。

随便尝试一下稍稍滥用 的写作风格。

展示一些好看的博客，我的奋斗目标就是把自己的博客也建设得这么好看。




更新日志

这是开始建站的日期，花了几天完成了一些简单的配置。

  • 博客使用 Hexo 搭建，部署在 GitHub Pages。
  • 主题使用 NexT.Mist。
  • 评论系统使用 Gitalk。
  • 使用不蒜子统计访客。
  • 使用 hexo-renderer-kramed + 渲染数学公式。
