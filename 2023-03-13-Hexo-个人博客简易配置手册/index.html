Hexo 个人博客简易配置手册

发表于 2023-03-13 更新于 2023-03-16 阅读次数：

Windows 下用 Hexo 和 GitHub 配置个人博客。

我的环境：

  • hexo: v6.3.0
  • npm: v6.14.16
  • Node.js: v12.22.12
  • git version 2.39.2.windows.1

一、准备工作

  1. 安装 Node.js :
    到 官网 安装Node.js，注意最新版只支持 Windows 8.1+，Win 7 用户请安装老版本。
    安装完成后在命令行输入 node -v 和 npm -v 以验证是否安装成功。

  2. 安装 git：
    要把网页上传到 GitHub，需要用到 git。
    安装完成后同样在命令行输入 git -v 验证是否安装成功。

  3. 注册 GitHub 账号并新建仓库：
    注册一个 github 账号。
    点击页面右上角 + 号选择 New repository 新建一个仓库，仓库名必须为 <你的 github 用户名>.github.io，例如我的仓库名为 azc-pkk.githu.io。

  1. 安装 hexo:
    在你喜欢的位置新建一个文件夹以存放你的博客文件，例如我的是 D:\Myblog。
    在该文件夹下单击右键选择 Git Bash here，会出现 git 的控制台窗口，输入以下指令： 然后输入 hexo -v 验证是否安装成功。
