流氓软件讨伐记

发表于 2023-03-18 更新于 2025-03-03

帮某老师删除了一下电脑里的流氓软件，记录一下作案过程。

那是在一个月黑风高的晚自习大课间，一个年轻的学生打开了老师的电脑。

IObit Uninstaller

首先祭出我们的屠龙宝刀 IObit Uninstaller，用于彻底删除流氓软件（下载地址）。

由于众所周知的原因官网下载比较慢，所以我先在家里下好后发给老师以节省时间。

CnSoftKiller

由大佬四海一叶秋制作，用于探测流氓软件。

人话：断网。

祭出武器只用了不到 2 分钟的时间，而敌人已向战场增派了两个团的兵力。两个陌生的图标出现在桌面：是新的流氓软件！

为了防止敌人越打越多，断网是及其必要且有效的措施。当然这里的断网肯定不是简单粗暴的拔网线，我们可以在电脑上完成简单优雅的断网：

打开网络和共享中心 -> 更改适配器设置 -> 选中正在工作的连接 -> 禁用此网络设备

至此，我们成功的封锁了战场，接下来，让我们开始收割流氓软件：

实习两步走战略：第一步，发现敌人；第二步，Kill it！

我们反着来，先学习使用屠龙宝刀：

打开 IObit Uninstaller 软件右上角工具选择强制卸载，把流氓软件所在文件路径复制上去即可。

有的软件自带卸载程序，请确定选择的是卸载而非修复或继续使用。

那么接下来就是要发现敌人，我使用了以下三种方法：

使用 SoftCnKiller

打开软件所在目录，下载更新。

打开 SoftCnKiller.exe，它可以帮我们扫描电脑中的流氓软件。扫描完成后全选并处理选中（图片仅供参考，虽然没有什么参考意义）。

然后文件夹里会出现一个 log 文件，标注了流氓软件所在位置（注意有可能重复标记）。

SoftCnKiller 还有一个弹窗定位的功能，但我并没有用到，所以请自行摸索。

Windows 任务管理器

按下 Ctrl+Alt+Delete 快捷键，启动任务管理器，右键一下你觉得是流氓软件的进程，可以找到它的位置。

试了一下没有发现流氓软件，可能看得见的都已经被 SoftCnKiller 扫描完了吧。

老师电脑里的流氓软件比我想象中多了好多，花了近一个小时才清理完，极大的超出了我的预期，导致我没有写完圆锥曲线 QAQ。

回到教室没多久就放学了，又一个晚自习被水过去了。

没有收到老师的后续反馈，想来清理的效果应该还好。

警惕流氓软件，尽量少在非官方网站下载程序，坚决不点开 P2P 下载器，遇事不决赶紧断网，坚决不使用2345极速浏览器等臭名昭著的国产流氓软件。
