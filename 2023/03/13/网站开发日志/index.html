网站开发日志

发表于 2023-03-13 更新于 2025-03-03

普普通通的开发日志。

前言

本来想写一篇用 搭建个人博客的教程的，但是看了很多大佬的博客后深刻的感受到了自己的弱小，所以改成了网站建设记录。

顺便尝试一下稍稍滥用 的写作风格。

贴一些好看的博客，希望未来我能有这个信心把自己的博客也贴上去。




更新日志

这是开始建站的日期，花了几天完成了一些简单的配置。

  • 博客使用 Hexo 搭建，部署在 GitHub Pages。
  • 主题使用 NexT.Mist。
  • 评论系统使用 Gitalk。
  • 使用不蒜子统计访客。
  • 使用 hexo-renderer-kramed + 渲染数学公式。
  • 使用路过图床。
