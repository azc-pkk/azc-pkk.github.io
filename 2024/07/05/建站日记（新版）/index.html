建站日记（新版）

发表于 2024-07-05 更新于 2024-08-04

2024 年高考后重建自己的博客，之后的更新都记录在这里。

前言

这个网站初建于 2023 年 3 月，当时会想到搓一个自己的个人博客是因为参加 NOIP 被打击到了，就放弃了参加省选（现在想来有点遗憾，应该去看看的），然后躺了几个月后重上洛谷，看到搭建博客的教程，就想着自己也来搭一个好了。

之前的博客从 6 月多开始就没有再更新了，现在高考完了，突然想起我的小网站，又正好换了新电脑，于是重新开始，重建我的博客。

首先确立写作路线，我想把所有博客分为三大类，分别是技术类，感想类和文艺类。

  • 技术类：包括学习笔记，题解，环境配置等方面内容。
  • 感想类：包括读书笔记，电影观后感等方面内容。
  • 文艺类：自己写的诗歌散文小说之类的。

其次是版本号的确立，采用 X.X.X 的形式，发生主题、域名等重大改变时第一位加一，增加较大功能第二位加一，修复 Bug、细微的功能增加或改变第三位加一。

此外，建站时间从今日（2024/7/5）开始重新计算。

Ver 1.0.0

2024/7/5 更新。

简略的重新配置了一下，比上次花的时间少了很多。

  • 依旧采用 Hexo 搭建，部署在 Github Pages。
  • 主题依旧是 NexT.Mist。
  • 依旧使用 hexo-filter-mathjax + hexo-renderer-kramed + 渲染数学公式。（参考hexo博客next主题添加对数学公式的支持）
  • 非常幸运的是，路过图床依旧能用。
  • 恢复了 订阅功能。（how to use?）
  • 依旧使用 ZCOOL KuaiLe 字体，好看。
  • 增加了本站运行时间

配置过程中遇到一个奇怪的 Bug：从之前电脑上复制下来的文章在博客首页显示为斜体，而文章页面显示正常。

对比图：


经过仔细对比，发现我的 md 文件第一行是空行，而 Hexo 自带的 md 文件没有空行，于是尝试删除该空行。删除后确实不再显示为斜体，但 Front-matter 中的内容显示了出来，文章名也显示为“未命名”。

尝试写了一篇新博客，发现没有显示问题，于是决定不使用原来的文件，而是新建文件，复制内容。结果正常了。

但是当我把所有博客复制上来后，这个 Bug 又出现了，于是我决定一篇一篇复制，复制一篇看一下效果。

经过深度排查后，我发现错误原因在于我在某文件中写了这样一句话：

相信会 html 的小伙伴都已经看出来了，这是标签未闭合导致的错误。。。而这篇文章刚好是我网站重建前发布的最后一篇文章，这就造成了“新建的文件没问题，复制的文件有问题”的假象。此外，在修复该 Bug 是，我发现 VS Code 中 md 预览已经明确揭示了该问题（整篇文章都是斜体），而我完全没有注意。。。

至于这个为什么在我原来的网站上能正常显示，我也不知道。。。

好了，修 Bug 环节到此结束，接下来是之后准备加入的功能：

  • Local Search
  • 评论区
  • 分享
  • 优化文章链接
  • 访客数量统计
  • 分类和标签
