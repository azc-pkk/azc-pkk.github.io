SoftCnKiller + IObit Uninstaller 删除流氓软件

发表于 2023-03-18 阅读次数：

这篇文章提供一种较彻底的删除流氓软件的方法。

准备工具

SoftCnKiller

SoftCnKiller 是大佬四海一叶秋制作的一个软件，其下载与使用方法见其博客。

IObit Uninstaller

用于彻底卸载软件。此为下载地址，由于墙的原因国内下载可能有困难，请自行解决。

开始

Step 1. 发现流氓软件

删除流氓软件的第一步当然是发现流氓软件，其方法有如下三种。

使用 SoftCnKiller

双击打开图示 bat 文件下载更新。

打开 SoftCnKiller.exe，该程序会自动扫描你电脑中的流氓软件。扫描完成后全选并处理选中（我已经清理过电脑里的流氓软件所以什么都没有发现）。

扫描后你的文件夹里会出现一个 log 文件，里面标注了你电脑中的流氓软件及其所在文件夹。

SoftCnKiller 还有一个弹窗定位的功能，打开 弹窗定位.exe，将应用 UI 中的靶心图标拖至广告弹窗处即可为追踪弹窗程序的文件路径。

Windows 任务管理器

按下 Ctrl+Alt+Delete 快捷键，选择启动任务管理器，右键可能是流氓软件的进程名，可以找到其所在文件路径。

Step 2.删除流氓软件

借助 IObit Uninstaller 彻底删除流氓软件，打开软件右上角工具选择强制卸载，将刚才找到的流氓软件路径复制上去即可。
